# Brain Tumor Final

This project uses a Decision Tree model to predict and analyze brain tumor conditions based on a structured dataset. It aims to assist in early identification and classification of tumor cases using machine learning.

## Features
- Decision Tree-based classification
- Interactive API for predictions
- Jupyter Notebook for data exploration and model training

## Dataset
The dataset (brain_tumor_dataset.csv) contains features relevant to tumor diagnosis. It is preprocessed and used to train the model.
